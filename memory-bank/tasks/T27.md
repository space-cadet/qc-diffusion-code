## Description
Rewrite the Random Walk simulation page around a pure WebGL renderer and a non-tsParticles control path while preserving the original physics engine’s strategy system. The task began as a clean-room `PhysicsEngineV2` rewrite, then pivoted to an adapter architecture where `WebGLRendererV2` is driven by the original `PhysicsEngine` through `useOriginalPhysicsEngine.ts`.

## Acceptance Criteria
- [x] Particles render through pure WebGL without tsParticles
- [x] V2 page mounts and uses the original physics engine through an adapter
- [x] `Initialize`, `Start`, `Pause`, and `Reset` affect the live engine
- [x] Density panel is present on the V2 page
- [x] Initial distribution selector is wired to actual particle initialization
- [x] Random walk page scrolls to lower panels on smaller viewports
- [x] Floating observables panels restored on the V2 page
- [x] Collision counts visible in V2 parameter panel status
- [ ] Graph mode parity restored on the V2 page
- [ ] Strategy dropdown only exposes implemented strategies or gains real `levy` / `fractional` implementations

## Implementation Details
See: `memory-bank/implementation-details/t27-clean-architecture-rewrite.md`

The current architecture is:
- `RandomWalkSimV2.tsx` owns layout and panel composition
- `ParticleCanvasV2.tsx` owns the render/update loop and emits live stats
- `useOriginalPhysicsEngine.ts` adapts the legacy engine, parameter manager, and initialization logic to the V2 WebGL path
- `WebGLRendererV2.ts` handles pure WebGL particle rendering

The main fixes landed after the original adapter swap:
- `T27a`: TypeScript/Vercel build strictness fixes
- `T27b`: Original engine integration
- `T27c`: Frozen-particle fix via finite initial collision times
- `T27d`: Strategy selector restoration in the V2 parameter panel
- `T27e`: Strategy propagation from UI state into the engine
- Additional parity fixes in this session: control wiring, visible motion scaling, live stats propagation, scroll restoration, density panel restoration, and initial distribution wiring restoration
