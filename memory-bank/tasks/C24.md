**Key Quantum Walk Mechanics**:
1. State: |ψ⟩ = Σ_x Σ_c |x⟩|c⟩ (position ⊗ coin space)
2. Evolution: U = S(C ⊗ I) where C = Hadamard coin, S = conditional shift
3. Measurement: Born rule P(x) = Σ_c |⟨x,c|ψ⟩|²
4. Initial state: |ψ₀⟩ = |0⟩|↑⟩ (position 0, coin up)

**Simplicity Focus (KIRSS)**:
- Start with minimal UI: parameter sliders + single plot
- Defer momentum space, comparison panels, advanced observables
- Hard-code Hadamard coin initially
- Position space only (not phase space)
